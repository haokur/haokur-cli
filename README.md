
### 1.运行

> 未发布 npm 包,仅支持本地安装.

1.1 下载源码

```
git clone https://github.com/haokur/webflow-cli.git --depth 1
```

1.2 安装本地的全局

```
npm install . -g
```

1.3 创建软链接,支持本地代码修改

```
npm link 
```
